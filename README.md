# Roberto Turbay Portal (Site Escolar)

Site institucional para a Escola Roberto Turbay — versão em desenvolvimento.

## Objetivo
Projeto criado para disponibilizar informações institucionais, documentos de matrícula, calendário e contato para pais e visitantes.

## Como rodar localmente
1. `npm install`
2. `npm run dev` (ou `pnpm dev` / `yarn dev`)

## Deploy
Este projeto é preparado para deploy no Vercel. Basta importar o repositório no Vercel e usar as configurações detectadas automaticamente.

## Estrutura principal
- `src/pages/` — páginas (informacoes.astro, localizacao.astro, documentos.astro, calendario.astro)
- `src/components/` — Header, Footer e componentes reutilizáveis
- `public/` — imagens e PDFs públicos
- `src/styles/global.css` — estilos globais
